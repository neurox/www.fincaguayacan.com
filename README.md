# Finca Guayacán - Landing Page

Static landing page for Finca Guayacán - Productos Orgánicos.

## Tech Stack
- Astro 4.x
- Tailwind CSS
- GitHub Pages

## Development

```bash
npm install
npm run dev
```

## Deployment

Auto-deploys to GitHub Pages on push to `main` branch.
